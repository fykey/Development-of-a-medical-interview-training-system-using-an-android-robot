# Development-of-a-medical-interview-training-system-using-an-android-robot
研究内容．人工筋肉をつかって，実際に表情が動くロボットを使った，医療面接訓練の開発を行っている．

診察動画を取って，OpenFaceのFeatureExtractionのアプリで，動画上の特徴点の変位や，そこから算出されるパラメータを取得．
